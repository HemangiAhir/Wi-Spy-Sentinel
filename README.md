\### Key Features



| Feature                        | Description |

|--------------------------------|-------------|

| \*\*Live Scanning\*\*              | Real-time Wi-Fi capture via Flipper Zero + ESP32 |

| \*\*Evil Twin Detection\*\*        | Detects multiple APs broadcasting the same SSID |

| \*\*Hidden Network Discovery\*\*   | Reveals networks that don't broadcast their name |

| \*\*Suspicious SSID Detection\*\*  | Flags "Free WiFi", "xfinitywifi", setup pages, etc. |

| \*\*Open Wi-Fi Warning\*\*         | Highlights unencrypted networks |

| \*\*Threat Scoring Engine\*\*      | Classifies networks as Safe / Low / Medium / High risk |

| \*\*Location-Based Comparison\*\*  | Compare home vs cafe vs airport vs hotel scans |

| \*\*Stunning Visual Reports\*\*    | Matplotlib-powered dashboards (pie charts, bars, signal strength) |

| \*\*CSV/JSON Export\*\*            | Full data export for further analysis |
